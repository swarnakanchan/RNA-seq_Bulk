Pipeline Ruhin RNA-seq: 

-5 Controls (Control-1_S1 to Control-1_S5), and 5 treatments (FLLL12-1_S6 to FLLL12-1_S10) are used in this analysis

-quality checking by fastqc (v0.12.1), 

-trimmomatic (v 0.39) was used for adapter trimming and quality control with parameters: ILLUMINACLIP:TruSeq3-PE.fa:2:30:10 SLIDINGWINDOW: 4:15 MINLEN:25

-Quality check using fastqc (v0.12.1) again performed after trimmomatic-based trimming

- Paired forward and reverse read sequences from all samples (Control and Treatment) were used further used for alignment by Hisat2 (version 2.2.1) using indexed human genome (GRCH38) 

-Alignment was followed by sorting of all bam files (controls and treatments) by sequence read names

-samtools fixmate-based processing was performed further for all samples

-Sorting of all bam files was further performed based on chromosomal positions

-Indexing of all bam files was further performed

-R (v. 4.4.1 ) and RStudio were used further to build a count table using summarizeOverlaps (features=exonsByGene, mode="IntersectionStrict", singleEnd=F, ignore.strand=T, fragments=F) from GenomicAlignments (v. 1.40.0)

-DESeq2 (v. 1.44.0 ) pipeline was used further for differential expression analysis of genes using the count matrix

- vst log transformation was performed to create sample similarity heatmap and PCA plot using pheatmap (v. 1.0.12), RColorBrewer (v. 1.1-3) libraries

-Sample similarity heatmap and PCA plot 

- Differentially expressed genes (significant genes) were marked by padj < 0.1. were marked as differentially expressed genes.

-Among which LFC > 0 (up): 6264 (19%) genes were marked as upregulated and LFC < 0 (down) : 6314 (19%) genes were marked as downregulated genes

- Symbol (Gene name) and Absolute fold change (2^(log2FoldChange)) columns were also included in the differentially expression analysis results
